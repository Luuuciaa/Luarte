# 🖌️ LUARTE - Artesanías en Yeso

Sitio web de Luarte, dedicado a la exhibición y venta de artesanías en yeso pintadas a mano. El objetivo es mostrar los productos de forma atractiva y facilitar el contacto con clientes.

---

## 📌 Descripción

Este proyecto es una página web estática con secciones informativas, galería de productos y elementos interactivos con JavaScript.

---

## 🚀 Funcionalidades

- Página principal con presentación de la marca  
- Sección "Nosotros"  
- Galería de imágenes de productos  
- Botón de contacto (WhatsApp)  
- Diseño visual atractivo  
- Navegación simple y clara  

---

## 🛠️ Tecnologías utilizadas
- HTML5
- CSS3
- JavaScript

## ⚙️ Cómo usar el proyecto
1. Descargar o clonar el repositorio
2. Abrir la carpeta del proyecto
3. Ejecutar el archivo index.html en el navegador

No requiere instalación adicional.
